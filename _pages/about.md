This site is a build log for **[tissue-bot](https://github.com/kurtc3b3/tissue-bot)** and related projects.

tissue-bot is an agentic system for collecting GitHub repository and issue data, storing it locally in SQLite, and analyzing or resolving issues with LangGraph agents.

Posts are written as we ship each layer of the stack — from the Python backend and FastAPI API, through the Next.js UI, agent integration, testing, and Docker deployment.
